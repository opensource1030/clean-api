## Lumen PHP Framework

## Quick Start

1. `git clone git@github.com:WirelessAnalytics/clean-api.git` 
2. `composer install`
3. `cd clean-api/public` 
4. Start PHP Server: `php -S localhost:4000`

Visit `localhost:4000` in your browser.
